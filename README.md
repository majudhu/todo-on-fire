# todo on Firebase

## Commands

### Setup

- `npm install -g firebase-tools`
- ~~`firebase init`~~ (create new empty firebase project)

### Serve Locally

- `firebase serve --only hosting`
- `firebase serve`
- `firebase serve --only functions,hosting`
- `firebase serve --only functions`

### Deploy

- `firebase login`
- `firebase deploy`
